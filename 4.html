The Anatomy of a Perfect Prompt

Module 02: Structure

The 6 Pillars

Essential Components

Persona / Role

บอกให้ AI สวมบทบาทเป็นใคร เพื่อกำหนดระดับความรู้และมุมมอง

e.g. Senior Java Dev

Context

ข้อมูลประกอบ สถานการณ์ หรือที่มาที่ไป ช่วยให้ AI เข้าใจบริบทงาน

e.g. Creating E-commerce App

Task

คำสั่งงานที่ชัดเจน สิ่งที่ต้องทำ คือเป้าหมายหลักของการ Prompt

e.g. Write a function

Constraint/Rules

ข้อจำกัดหรือกฎที่ห้ามละเมิด เพื่อควบคุมขอบเขตของผลลัพธ์

e.g. No external libraries

Output Format

รูปแบบผลลัพธ์ที่ต้องการ ง่ายต่อการนำไปใช้งานต่อ

e.g. Markdown, JSON

Example

ตัวอย่าง Input และ Output ที่อยากได้ (Few-shot) เพื่อให้เห็นภาพชัดสุด

e.g. Input: A -> Output: B
