Key Principles of Prompt Optimization

Module 02: Structure & Techniques

องค์ประกอบสำคัญของ Prompt ที่ดี

Essential Components

Persona / Role

บอกให้ AI สวมบทบาทเป็นใคร เพื่อกำหนดระดับความรู้และมุมมอง

e.g. "รับบทเป็น Senior Frontend Developer"

Context

ข้อมูลประกอบ สถานการณ์ หรือที่มาที่ไป ช่วยให้ AI เข้าใจบริบทงาน

e.g. "ได้รับคำสั่งให้พัฒนา E-commerce App ที่มีลูกค้าเป็นผู้บริโภคกลุ่มนักศึกษา..."

Task

คำสั่งงานที่ชัดเจน สิ่งที่ต้องทำ คือเป้าหมายหลักของการ Prompt

e.g. "เขียน Code หน้า Dashboard ตาม Design ที่ผมจะส่งให้"

Constraint/Rules

ข้อจำกัดหรือกฎที่ห้ามละเมิด เพื่อควบคุมขอบเขตของผลลัพธ์

e.g. "ใช้ภาษา JavaScript และ React.js เท่านั้น หากต้องการใช้ CSS ให้ใช้ Tailwind CSS เท่านั้น"

Output Format

รูปแบบผลลัพธ์ที่ต้องการ ง่ายต่อการนำไปใช้งานต่อ

e.g. "ขอเป็น Code JavaScript (ES6+) ที่มี Comment อธิบาย"

Example

ตัวอย่าง Input และ Output ที่อยากได้ เพื่อให้เห็นภาพชัดสุด

e.g. "ตัวอย่าง Code และ ตัวอย่างอธิบาย"

Advanced Techniques

Optimization

Chain-of-Thought

สั่งให้ AI คิดทีละขั้น (Step-by-step) เพื่อลดข้อผิดพลาด

e.g. Let's think step by step

Iteration

ทดสอบ -> ปรับปรุง -> ทำซ้ำเพื่อให้ได้ผลลัพธ์ที่ดีที่สุด

e.g. "ปรับแก้ Prompt จากผลลัพธ์รอบแรก"
