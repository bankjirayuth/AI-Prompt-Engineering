Module 01: Core Concepts

DEFINITION

ศาสตร์ในการออกแบบคำสั่ง (Input) เพื่อควบคุม AI ให้สร้างผลลัพธ์ (Output) ที่มีคุณภาพและตรงตามความต้องการของเรามากที่สุด

ROLE OF  PROMPT ENGINEERING  IN AI SYSTEMS

Control Output

ควบคุมรูปแบบ คุณภาพ และความถูกต้องของผลลัพธ์

Reduce Errors

ลด Hallucination จาก AI ด้วย Context(ข้อมูล) ที่ชัดเจน

Maximize Value

ดึงศักยภาพสูงสุดจาก AI

Knowledge: Hallucination คือ สร้างคำตอบ ข้อมูล หรือเนื้อหาที่ผิดพลาด ไม่เป็นความจริง หรือไม่สมเหตุสมผลขึ้นมาเอง

ผู้ใช้งาน AI ทั่วไป vs มีทักษะ AI Prompt Engineering

ไม่รู้ PE
✗ ถามกว้างๆ ได้คำตอบไม่ตรงจุด
✗ ต้องแก้ไข Output หลายรอบ
✗ เสียเวลาลองผิดลองถูก
✗ ใช้ AI ไม่เต็มศักยภาพ
มีทักษะ PE
✓ สั่งเฉพาะเจาะจง ได้ผลตรงใจ
✓ ได้ Output พร้อมใช้ทันที
✓ ทำงานเร็วขึ้น
✓ ดึงศักยภาพ AI ได้เต็มที่

What's the benefits?

เพิ่มความสม่ำเสมอ

ควบคุม Format และ Standard ผ่าน System Prompt

ลดเวลางานจำเจ

ประหยัดเวลาเขียน Boilerplate/Template, Unit Test หรือสรุปเอกสาร Technical ที่มีรูปแบบซ้ำๆ

ประโยชน์ต่อการพัฒนา

สามารถใช้ AI เป็น Pair Programmer ที่ช่วย Review และ Suggest Code หรือช่วยแก้ไข Error ที่เกิดขึ้น
