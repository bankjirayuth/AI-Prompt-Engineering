Best Practices

Version Your Prompts

เก็บ history ของ prompt แต่ละ version เพื่อเปรียบเทียบและ rollback ได้

A/B Test Variations

ทดสอบ prompt หลาย versions กับ input เดียวกัน เปรียบเทียบผลลัพธ์

Define Success Criteria

กำหนด metrics ที่ชัดเจน เช่น accuracy, format compliance, relevance

Document Learnings

จดบันทึกสิ่งที่เรียนรู้ว่า prompt แบบไหน work/ไม่ work กับ use case นั้นๆ

Pro Tip: เริ่มจาก prompt ง่ายๆ ก่อน แล้วค่อยๆ เพิ่มความซับซ้อนทีละ step
