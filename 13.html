Module 03: Practice

Your Turn

ใบงานสำหรับผู้เข้าร่วม

Situation

มี Requirement ยาวเหยียด 10 หน้า ขี้เกียจอ่าน อยากรู้แค่ว่าต้องทำ API อะไรบ้าง

Bad Prompt

"สรุปอันนี้ให้หน่อย [แปะ Text]"

Mission

เขียน Prompt ให้ AI สรุปสิ่งที่ต้องทำออกมาเป็น ข้อๆ ให้ชัดเจน และพร้อมนำไปใช้

Solution Guide

เฉลย / แนวทาง

Persona
รับบทเป็น System Analyst (SA)
Context
ได้รับเอกสาร Requirement จากลูกค้า (Business Requirement)
Task
ช่วยวิเคราะห์และสกัดรายการ API ที่ต้องพัฒนา (API List) จากข้อความนี้
Constraints

แสดงผลเป็นตาราง มีคอลัมน์:

  • ชื่อ API (Method + Path)
  • หน้าที่ (Description)
  • Input ที่จำเป็น
Output
