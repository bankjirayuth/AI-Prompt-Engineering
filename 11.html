Your Turn

ใบงานสำหรับผู้เข้าร่วม

Situation

มีฟังก์ชันตรวจสอบรหัสผ่าน isValidPassword(password) แต่ยังไม่มี Test Case เลย

Bad Prompt

"เขียน Test ให้หน่อย [แปะโค้ด]"

Mission

เขียน Prompt ให้ครอบคลุม เคสสำคัญๆ ไม่ใช่แค่เคสปกติ (Happy Path)

Solution Guide

เฉลย / แนวทาง

Persona
รับบทเป็น Senior Tester
Context
เรามีฟังก์ชัน isValidPassword ที่ต้องการ ความปลอดภัยสูง
Task
เขียน Unit Test Case สำหรับฟังก์ชันนี้
Constraints

ต้องครอบคลุม 3 กรณี:

  • รหัสผ่านถูกต้อง (มีตัวใหญ่, เล็ก, ตัวเลข, อักขระพิเศษ)
  • สั้นเกินไป (< 8 ตัว)
  • ขาดตัวเลขหรืออักขระพิเศษ
Output
ตารางอธิบาย Test Scenario (ยังไม่ต้องเขียนโค้ดจริง)
Expected Output
Case Input Expected
Valid "Pass123$" True
Too Short "Pa1$" False
